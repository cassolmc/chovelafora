Sítio Chove Lá Fora

Este jogo precisa ser instalado para jogar.

A instalação ainda não está disponível. Continue na página por alguns segundos (ou toque em algo) e o botão vai liberar.

Para instalar no iPhone/iPad:

  1. Toque no botão ↑ Compartilhar do Safari
  2. Escolha "Adicionar à Tela de Início"
  3. Abra o app instalado para jogar
Problemas para instalar?
  • No Android, use o Chrome: menu ⋮ → "Adicionar à tela inicial" ou "Instalar app".
  • Já instalou antes? O botão não aparece de novo — abra o jogo pelo ícone na tela inicial.
  • Aba anônima/privada não permite instalar.
  • No computador, procure o ícone de instalação na barra de endereço do Chrome/Edge.
